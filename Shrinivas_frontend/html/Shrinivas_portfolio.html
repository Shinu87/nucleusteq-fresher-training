    • Converted book content into vector embeddings and stored them in Pinecone for fast retrieval.
    • Used LangChain and Gemini 1.5 to generate answers based on retrieved medical information.
    • Implemented RAG to provide accurate and context-based responses.

  2. Resume Classification using Hybrid NLP Models

    Technologies Used : Python, TF-IDF, BERT, NLTK, Gemini API (GitHub)

    • Built a system to classify resumes into job categories using NLP.
    • Processed text using NLTK (tokenization and stop-word removal).
    • Used TF-IDF and BERT embeddings to convert text into numerical vectors.
    • Handled imbalanced data using SMOTE and achieved 71.2% accuracy.
    • Integrated Gemini API to generate resume improvement suggestions.
